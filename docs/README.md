# PHPSprinkles Docs

This directory is intentionally split by document role.

- [roadmap.md](roadmap.md): current source of truth for what comes next and in
  what order
- [architecture.md](architecture.md): stable platform structure, layering, and
  ownership rules
- [plugin-development.md](plugin-development.md): where plugins belong and how
  to develop them
- [customization-cookbook-plan.md](customization-cookbook-plan.md): planned
  cookbook-style documentation system for framework customizations
- [implementation-plan.md](implementation-plan.md): historical early setup plan
- [platform-roadmap.md](platform-roadmap.md): superseded roadmap kept as a
  redirect

If you are not sure where to start, begin with [roadmap.md](roadmap.md) and
then read [architecture.md](architecture.md).
